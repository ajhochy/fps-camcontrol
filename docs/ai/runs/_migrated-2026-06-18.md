### 2026-05-18 — Round 7: DJI gimbal spec + research (no code)
Wrote `docs/dji-gimbal-spec.md` — full technical spec for adding DJI RS4 Pro support via a Raspberry Pi bridge. Researched DJI's RS SDK, ConstantRobotics/DJIR_SDK, Hibiki1020 RS3 Pro controller, ArduPilot DJI driver docs, and the Log-and-Replay project. **Key findings:**
- DJI's RS SDK officially covers RS4 Pro, RS3 Pro, RS4, RS5. Public, no NDA.
- USB-C control is **not** possible on any RS-series gimbal — DJI's USB-C is charging + firmware updates + camera passthrough only. CAN bus required.
- Cheapest hardware path: CANable Pro USB dongle + RSA pigtail (~$55). For permanent install: PiCAN3 HAT + PoE++ splitter (~$200).
- Bluetooth motion control: Ronin app uses it but protocol is closed and never reverse-engineered. Not viable.
- Recommended adapter: DJI's official SDK (or implement the CAN protocol from the Interface Diagram PDF if SDK is Windows-only).
